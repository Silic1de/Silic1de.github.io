用Gensim训练word2vec

在gensim中实现word2vec很简单。
gensim中封装了word2vec，doc2vec等模型，本文主要涉及word2vec的使用

模型

gensim官方文档

主要参数

  • sentences 预处理好的语料库，必须是可迭代的，可以是简单的列表或者迭代器
  • size 维度
  • window 窗口大小
  • min_count 最小出现次数，忽略频数低于它的单词（不计算词向量）
  • alpha 学习率
  • sg 训练算法 1 for skip-gram; otherwise CBOW
  • negative 负采样数量
  • workers 线程数

详细参考官方文档

训练

sentences为可迭代的对象，这里考虑两种情况

语料为列表

每行对应一个句子，完成分词

Python列表可以直接作为输入，但输入量很大时会占据大量内存

语料为文件（处理为迭代器）

对路径下的所有文件生效
这里使用了哈利波特原著共7部作为语料库

模型保存与加载

Word2Vec对象还支持online learning。
可以将更多训练数据传递给一个以及训练好的Word2Vec对象，继续更新参数。
